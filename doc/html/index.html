Introduction

Ce programme C est une reproduction du jeu de la vie.
Le programme ne fonctionne que dans un terminal, et nécessite d'utiliser des fichiers textes pour le configurer.

Structure de fichier

Le projet comporte plusieurs répertoires : un répertoire source, un répertoire pour l'exécutable et un répertoire pour la documentation doxygen.
Le répertoire source contient les fichiers sources du programme, ce sont ces fichiers qu'il faudra compiler si l'exécutable est supprimé.
Par défaut, le répertoire bin contient déjà l'exécutable.

Compilation du programme

Si, pour une quelconque raison, vous devez recompiler le programme, voici un makefile simple qui vous le permettra : 

srcdir = ./src/
bindir = ./bin/
SRC = $(wildcard $(srcdir)*.c)
OBJ = $(subst $(srcdir),$(bindir),$(SRC:.c=.o))
PROG = $(bindir)exe
CC = gcc -Wall -g

all : $(PROG)
$(PROG) : $(OBJ)
    @echo 'Compilation de l executable'
    $(CC) $^ -o $@ -lm
./bin/%.o : ./src/%.c
    @echo 'Compilation des fichiers objets'
    $(CC) -c $^ -o $@ -lm

À noter que ce makefile doit se trouver dans le répertoire parent de src et bin.
De même, faites bien attention à bien placer des tabulations là où elles sont nécessaires, makefile étant très sensible aux espaces. Si le message d'erreur "missing separator. Stop" apparaît, replacer toutes les tabulations manuellement.

Structure du fichier de configuration

Le fichier doit s'ouvrir sur deux nombres, représentant la longueur et la largueur de la matrice.
Ensuite se trouve les valeurs de la matrice.
Puis nous trouvons un nombre répresentant les tours à effectuer.
Enfin se trouve soit 0 soit 1, nombe qui demande au programme d'utiliser une matrice simple ou une matrice torique (respectivement).

Attention : un fichier mal structuré ou mal configuré fera planter le programme.

Execution du programme

Le programme s'execute en utilisant en argument le nom (ne pas oublier l'extension) ou l'adresse (relative ou absolue) du programme.
Quelques exemples (avec le fichier placé dans le répertoire parent et l'exécutable dans le répertoire bin): 

- Depuis le répertoire parent : ./bin/exe fichier.txt
- Depuis le répertoire bin : ./exe ../fichier.txt
